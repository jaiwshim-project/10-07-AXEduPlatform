🏠 AX교육플랫폼 소개

AX교육플랫폼은 AI 전환(AX) 전문가를 양성하는 한국 최초의 통합 생태계 플랫폼입니다. 교육 → 인증 → 커뮤니티 → 실전 프로젝트로 이어지는 완전한 AX 전문가 성장 경로를 제공합니다.

플랫폼의 핵심 가치

📚 체계적인 AX 교육

입문부터 심화까지 50개 강의. Gemini API, Supabase, 바이브코딩 등 실전 중심 커리큘럼.

🏅 공신력 있는 인증

AX Practitioner → Expert → Architect 3단계 인증. 업계 공인 전문가 자격 부여.

🤝 활발한 커뮤니티

2,847명 AI 전환 전문가 네트워크. Q&A, 프로젝트 모집, 스터디 모임.

🚀 실전 프로젝트 매칭

247개 기업과 AX 전문가 매칭. 실제 수익을 창출하는 프로젝트 참여 기회.

ax-education-platform.vercel.app
🖥️
메인 랜딩 페이지 스크린샷
Hero 섹션, 주요 기능 소개, CTA 버튼

▲ AX교육플랫폼 메인 화면. 학습 경로, 주요 기능, 전문가 통계를 한 눈에 확인할 수 있습니다.

이런 분들에게 적합합니다

  • 입문자 AI 코딩 경험이 없어도 시작할 수 있습니다. 바이브코딩으로 2시간 만에 첫 AI 앱을 완성!
  • 개발자 기존 개발 역량에 AI를 더해 AX 전문가로 업그레이드. 연봉 2배 전환 레퍼런스 다수.
  • 기업 임원 조직의 AI 전환을 이끌 전략과 실행 역량을 습득. 검증된 전문가와 즉시 협업 가능.
  • 강사/멘토 AX 강의를 개설하고 수강료 수익을 창출. 멘토링으로 추가 소득 가능.

👤 회원가입 / 로그인

AX교육플랫폼은 Supabase Auth를 기반으로 안전한 인증을 제공합니다. 이메일 또는 소셜 로그인(Google, GitHub)으로 30초 만에 가입할 수 있습니다.

회원가입 방법

1
우측 상단 "무료 시작" 버튼 클릭
네비게이션 바의 주황색 "무료 시작" 버튼을 클릭하면 회원가입 페이지로 이동합니다.
2
가입 방법 선택
이메일 직접 가입 또는 Google/GitHub 소셜 로그인 중 선택할 수 있습니다. 소셜 로그인이 가장 빠릅니다.
3
프로필 설정
이름, 직업/역할, 관심 분야를 입력하면 맞춤 강의를 추천받을 수 있습니다. (선택 사항)
4
이메일 인증 (이메일 가입 시)
가입하신 이메일로 인증 링크가 발송됩니다. 링크 클릭 후 로그인하세요.
🔐
회원가입 / 로그인 화면 스크린샷
이메일 입력, 소셜 로그인 버튼, 이용약관 동의

▲ 회원가입 페이지. Google 또는 GitHub 계정으로 원클릭 가입이 가능합니다.

⚠️
보안 주의: 비밀번호는 8자 이상, 영문+숫자+특수문자 조합을 권장합니다. 플랫폼은 비밀번호를 평문으로 저장하지 않습니다 (bcrypt 암호화).

📝 과제 제출

각 강의 모듈 완료 후 실습 과제를 제출합니다. AI가 1차 피드백을 제공하고, 강사가 최종 리뷰를 완료합니다.

과제 제출 방법

1
GitHub 저장소 생성
과제 코드를 올릴 GitHub 공개 저장소를 생성합니다. 플랫폼에서 자동 연동 지원.
2
코드 작성 및 커밋
바이브코딩 방식으로 과제를 완성하고 GitHub에 푸시합니다.
3
URL 제출
GitHub URL과 배포된 서비스 URL을 과제 제출 폼에 입력합니다.
4
AI + 강사 리뷰
AI가 48시간 내 1차 코드 리뷰, 강사가 72시간 내 최종 피드백을 제공합니다.

🏅 전문가 인증 단계

AX교육플랫폼은 3단계 전문가 인증 시스템을 운영합니다. 각 단계는 이론 시험 + 실전 포트폴리오로 구성되며, 업계 공신력 있는 자격을 부여합니다.

🎯

AX Practitioner

입문 → 중급

  • ✓ 과제 통과율 70% 이상
  • ✓ 온라인 객관식 시험 (100점 만점, 70점 이상)
  • ✓ 포트폴리오 프로젝트 1개 제출

AX Expert

중급 → 고급

  • ✓ 멘토 추천서 1부

📋 시험 응시 방법

인증 시험은 온라인으로 진행됩니다. 언제든지 응시 가능하며, 탈락 시 2주 후 재응시 가능합니다.

시험 신청 방법

1
마이페이지 → 인증 탭 이동
좌측 대시보드에서 "인증 관리" 메뉴를 클릭합니다.
2
응시 자격 확인

3
시험 시작
객관식 50문항, 90분 제한. 안정적인 인터넷 환경에서 응시하세요.
⚠️
시험 부정행위 방지: 시험 중 별도 탭 전환 3회 이상 시 자동 종료됩니다. 신뢰할 수 있는 환경에서 응시하세요.

❌ 이렇게 하지 마세요

FAQ

최소 프로젝트 예산이 얼마인가요? ▼
최소 300만원부터 의뢰 가능합니다. 자동화 프로젝트의 경우 더 저렴하게 진행될 수 있으며, 구체적인 요구사항에 따라 견적이 달라집니다.
프리랜서와의 차이점은 무엇인가요? ▼
모든 전문가는 AX 인증 보유자로, 역량이 검증되어 있습니다. 또한 전담 PM이 품질을 관리하고, 결과물 미달 시 환불 또는 재작업을 보장합니다. 프리랜서 플랫폼과 달리 AX 특화 전문가만 등록되어 있습니다.
NDA (비밀유지계약)는 어떻게 처리되나요? ▼
모든 프로젝트에 표준 NDA가 자동으로 체결됩니다. 추가 보안 조항이 필요하시면 계약 전 요청하시면 됩니다. 전자 서명으로 빠르게 처리됩니다.
